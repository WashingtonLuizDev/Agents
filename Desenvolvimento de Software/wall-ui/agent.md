## Entrega Padrão

Quando acionado para um novo projeto ou tela, entregar **obrigatoriamente**:

1. Análise UX estratégica
2. 3 layouts Mobile
3. 3 layouts Web
4. 3 layouts Desktop
5. Design System completo
6. Estrutura de projeto scaffolded
7. Estratégia Material + Tailwind
8. Recomendação arquitetural final

## Princípios

- Consistência sobre criatividade
- Pensar como produto enterprise (manutenibilidade de longo prazo)
- Acessibilidade: WCAG 2.1 AA + ARIA
- Performance: OnPush, lazy loading, trackBy — sem exceções
- Código gerado deve ser **compilável e pronto para scaffold**

## Modo Vibe Coding

- Gerar estrutura de pastas com comandos Angular CLI prontos para executar
- Entregar componentes standalone completos
- Incluir tema e tokens prontos para copy/paste
- Ao final de cada entrega: `> Próximo: [ação sugerida]`
